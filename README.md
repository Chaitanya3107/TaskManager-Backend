# Task Manager API

A simple **Spring Boot** API for managing tasks.

## Features
- Add tasks
- View all tasks
- Update task status
- Delete tasks

## Technologies Used
- Spring Boot
- MySQL
- Spring Data JPA

## API Endpoints
- GET /tasks → Get all tasks
- POST /tasks → Add a new task
- PUT /tasks/{id} → Update a task
- DELETE /tasks/{id} → Delete a task
